# 🗺️ Chicago Crime Hotspot Predictor

## Overview
A machine learning web app that predicts and visualizes 
crime hotspots in Chicago using real crime data!

## 🌐 Live Demo
https://ml-models-phegmw5bamxqrydktfm5yq.streamlit.app/
## 🔥 Features
- Interactive heatmap of crime hotspots
- Filter by Year and Crime Type
- Real Chicago crime data (2012-2017)
- ML model predicts crime patterns
- Top 10 crime statistics chart

## 🛠️ Tech Stack
- Python
- Streamlit (Web App)
- Folium (Interactive Maps)
- Random Forest (ML Model)
- Pandas (Data Processing)

## 📊 Dataset
Chicago Crimes Dataset from Kaggle
- 500,000+ real crime records
- 2012 to 2017 data

## 🚀 How to Run Locally
pip install -r requirements.txt
streamlit run app.py

## 📁 Project Structure
├── app.py                    # Main web app
├── crime_model_small.pkl     # ML Model
├── label_encoder.pkl         # Label Encoder
├── chicago_crimes_sample.csv # Sample Data
└── requirements.txt          # Dependencies

## 🎯 ML Model
- Algorithm: Random Forest Classifier
- Features: Latitude, Longitude, Year
- Predicts: Crime Type
